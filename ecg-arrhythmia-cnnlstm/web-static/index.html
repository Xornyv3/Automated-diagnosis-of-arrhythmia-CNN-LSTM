Automated diagnosis of arrhythmia using combination of CNN and LSTM techniques with variable length heart beats
by Hala Azzouz — 3rd year GBM student at ENSAM Rabat.

Overview

This project preprocesses ECG records (MIT‑BIH), extracts 1000‑sample beats centered on R‑peaks (500 samples before and after), applies per‑segment z‑score normalization, and trains a Conv1D → LSTM model (Keras). Training uses stratified 10‑fold cross validation with class weights and optional augmentation/bandpass.

Notes & interpretation

  • High specificity across classes indicates few false positives overall.
  • Lower PPV for APB and PVC highlights class imbalance; targeted augmentation can help.
  • The CNN extracts morphology while the LSTM models temporal structure within each beat window.
